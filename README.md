# CareerRestart AI

CareerRestart AI is an Agentic AI application built using Amazon Nova that helps professionals returning from career breaks restart their careers.

## What it does
Users upload their resume and paste a job description. The AI analyzes:
- Missing skills
- What to learn
- A personalized career roadmap

## Tech Stack
- Streamlit
- Amazon Nova (AWS Bedrock)
- Python

## How to run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Run the app  
   `streamlit run app.py`
